2048

  • 2048 is a single-player sliding block puzzle game by Italian web developer Gabriele Cirulli.
  • 2048 is played on a gray 4Ã—4 grid, with numbered tiles that slide smoothly when a player moves them using the four arrow keys.
  • Every turn, a new tile will randomly appear in an empty spot on the board with a value of either 2 or 4.
  • Use keyboard arrows if you are playing from a Computer.
  • Use given navigation keys if you are playing from a Phone.
